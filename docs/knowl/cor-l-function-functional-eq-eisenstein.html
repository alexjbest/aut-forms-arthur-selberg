Corollary1.68
\begin{equation*} \Lambda(s) = \pi^{-s} \Gamma(s) \zeta(2s) \tilde a_0 (s-1) \end{equation*}

we showed

\begin{equation*} \Lambda(s) = \int_{\SL_2(\ZZ) \backslash \HH} \phi(s) E(z; s) \diff \mu (z)\text{.} \end{equation*}

(So we can find a functional equation and analytic continuation for from the corresponding properties of the Eisenstein series.)

  • \(\Lambda(s)\) has analytic continuation.
  • \(\Lambda(s)\) has functional equation \(s\leftrightarrow s-1\text{.}\)
  • \(\Lambda(s)\) has poles only at \(s= 0,1\text{.}\)
  • \begin{equation*} \Res_{s=1} = \int_{\SL_2(\ZZ) \backslash \HH} \phi(s) \diff \mu(z) \end{equation*}
