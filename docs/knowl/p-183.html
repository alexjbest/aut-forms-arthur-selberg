Let \(f(z)\ne 0\) be a cusp form of weight \(k\) which is an eigenfunction for all of the Hecke operators with eigenvalue \(n^{1-k/2} \lambda(n)\) i.e.

\begin{equation*} T_n(f)(z) n^{1-k/2} \lambda(n) f(z) \forall n \end{equation*}

let

\begin{equation*} f(z) = \sum_{n=1}^\infty a_n e(nz) \end{equation*}

be its fourier expansion and

\begin{equation*} L(s,f) = \sum_{n=1}^\infty \frac{a_n}{n^s}\text{.} \end{equation*}

Then

  1. \begin{equation*} a_1 \ne 0 \end{equation*}
  2. If \(a_1 =1\) then \(a_n = \lambda(n) \forall n\text{.}\)
  3. If \(a_1 = 1\) then \(\lambda(mn) = \lambda(m)\lambda(n)\) for all \((m,n) = 1\text{.}\)
  4. If \(a_1 = 1\) then
    \begin{equation*} L(s,f) = \prod_p(1-a_pp^{-s} + p^{k - 1- 2s})^{-1}\text{.} \end{equation*}
