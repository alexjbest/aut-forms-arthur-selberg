Let \(\phi\colon \HH \to \CC\) be automorphic with respect to \(\SL_2(\ZZ)\text{,}\) with fourier expansion

\begin{equation*} \phi(z) = \sum_{n=-\infty}^\infty a_n(y) e(nx),\,\text{where}\,a_n(y) = \int_0^1 \phi(x+iy) e(-nx) \diff x\text{.} \end{equation*}

If \(\phi(x+iy) = O(y^{-N})\) for all \(N \gt 0\text{.}\)

\begin{equation} \int_{\SL_2(\ZZ) \backslash \HH} \phi(z) E_3( z; s)\diff \mu(z) = \pi^{-s} \Gamma(s) \zeta(2s) \tilde a_0(s-1)\tag{1.4} \end{equation}

where \(\phi(z) = \sum_{n\in \ZZ} a_n(y) e(nx)\text{.}\)
