Proof
  1. \(\ints_K\) is a full rank lattice in \(\prod_{v|\infty} K_v = V\) (Note for \(\lb K: \QQ \rb = N\) then \(V \simeq \RR^{r_1} \times \CC^{r_2} \simeq \RR^N\)).
  3. Obvious
  4. Partial fractions.
  5. \begin{equation*} \mu(K\backslash \adeles) = \mu (K\backslash V) \mu(\pi\ints_v) = 1\text{.} \end{equation*}
  6. Since \(\adeles_K / K\) is compact, \(\widehat{\adeles_K/K}\) is discrete. Further \(K \subseteq K^\perp\text{.}\) In fact \(K^\perp /K\) is a finite vector space since
    \begin{equation*} K^\perp/K \hookrightarrow \widehat \adeles_K/K \simeq \adeles/K \end{equation*}
    which is compact, hence \(K^\perp = K\text{.}\)
